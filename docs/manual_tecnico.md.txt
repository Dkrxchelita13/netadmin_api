# Manual Técnico - NetAdmin API

## 1. Nombre del proyecto

**NetAdmin API – Sistema de Inventario, Monitoreo y Administración de Red**

## 2. Objetivo del sistema

Desarrollar una aplicación en Python que permita recabar información básica de una red, almacenar dispositivos en un inventario estructurado y administrar la información mediante una API REST.

El sistema permite registrar dispositivos manualmente, consultar el inventario, buscar dispositivos por dirección IP, actualizar registros, eliminar dispositivos obsoletos, escanear una red local simulada y exportar la información en formatos JSON, YAML y XML.
